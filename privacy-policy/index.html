1. 简介 1. Introduction

重要 IMPORTANT 本隐私政策描述了GitHub文件加速服务（以下简称"本服务"）如何处理您的信息。本服务提供GitHub文件的加速下载和查看功能，无需用户注册或登录即可使用。 This Privacy Policy describes how GitHub File Accelerator (hereinafter referred to as "the Service") handles your information. The Service provides accelerated downloading and viewing of GitHub files without requiring user registration or login.

请注意，虽然本服务本身不要求登录，但我们使用了第三方评论系统 Waline，该系统提供了登录和评论功能。当您使用评论功能时，Waline会收集相关信息。 Please note that while the Service itself does not require login, we use a third-party commenting system Waline which provides login and commenting functionality. When you use the commenting feature, Waline collects relevant information.
